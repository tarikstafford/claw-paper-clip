<task type="auto">
  <name>Task 2: Implement CompactionService with token counting, threshold logic, and prompt builder</name>
  <files>server/src/services/compaction.ts</files>
  <action>
Create `server/src/services/compaction.ts` as a factory function `compactionService(db, anthropicClient)` that returns an object with these methods:

**countMessageTokens(body: string, model: string): Promise&lt;number | null&gt;**
- Call `anthropicClient.messages.countTokens({ model, messages: [{ role: "user", content: body }] })` and return `result.input_tokens`.
- Wrap in try/catch. On failure, log error and return `null` (best-effort per COMP-01 design).

**buildThreadPrompt(threadId: string, model: string): Promise&lt;{ prompt: string; wasCompacted: boolean; tokensBefore: number; tokensAfter: number }&gt;**
- Load all messages for thread ordered by createdAt asc using `db.select().from(chatMessages).where(eq(chatMessages.threadId, threadId)).orderBy(asc(chatMessages.createdAt))`.
- Sum `tokenCount` for all messages (skip null — use character estimate `Math.ceil(body.length / 4)` as fallback for messages with null tokenCount).
- Check threshold: `totalTokens >= Math.floor(getContextWindow(model) * 0.55)`.
- If below threshold: format all messages as verbatim turns and return `{ prompt: formatted, wasCompacted: false, tokensBefore: totalTokens, tokensAfter: totalTokens }`.
- If above threshold:
  - Split: `toSummarize = messages[0 .. length - DEFAULT_VERBATIM_TURNS - 1]`, `verbatim = messages[length - DEFAULT_VERBATIM_TURNS ..]`. If messages.length <= DEFAULT_VERBATIM_TURNS, no compaction needed.
  - Call `summarizeMessages(toSummarize)` via Anthropic Messages API using `claude-haiku-3-5` with the summarization system prompt from research.
  - Record audit event: insert into `chatCompactionEvents` with compactedMessageCount, summaryTokenCount (count tokens of summary), tokenCountBefore, tokenCountAfter, summaryText, model.
  - Format prompt: `<conversation_summary>\n{summary}\n</conversation_summary>\n\n<recent_messages>\n{formatted verbatim messages}\n</recent_messages>`.
  - Return `{ prompt, wasCompacted: true, tokensBefore, tokensAfter }`.

**Helper: formatMessages(messages)** — Format each message as `[{senderType}]: {body}` separated by newlines.

**Helper: toAnthropicMessages(messages)** — Map senderType to role (`agent` -> `assistant`, all others -> `user`). Merge consecutive same-role messages by concatenating content with `\n` to avoid Anthropic alternating-role constraint violation.
